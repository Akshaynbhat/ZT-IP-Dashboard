Title: Zero Trust Architecture: Enhancing Cybersecurity in Enterprise Networks
Authors: Engr. Tahir Bashir
Year: 2024

Problem Statement:
Traditional perimeter-based security models are insufficient to protect modern enterprise networks from advanced cyber threats, cloud security risks, and unauthorized access.

Summary:
This paper proposes Zero Trust Architecture (ZTA) to improve enterprise cybersecurity through continuous authentication, least-privilege access, micro-segmentation, and real-time monitoring.

Limitation:
Does not provide a real-time dynamic trust scoring mechanism for users and devices.

Solution:
Implement Zero Trust Architecture (ZTA) with continuous verification, access control, network segmentation, encryption, and AI-assisted threat detection.

Why This Solution is Used:
ZTA follows the "Never Trust, Always Verify" principle, ensuring every user and device is continuously authenticated, thereby reducing unauthorized access and security breaches.
