Insider threats targeting valuable software Intellectual Property (IP) represent a critical security challenge for modern software development environments. Traditional static access controls and perimeter defense mechanisms fail to detect authenticated adversaries, while existing machine learning security tools lack dynamic, explainable trust estimation at the decision point to guide runtime access policy decisions. To address this, this work proposes a Zero Trust-aligned framework that integrates a two-stage hybrid machine learning pipeline—combining unsupervised Isolation Forest anomaly scoring with a supervised Random Forest risk classifier—into an adaptive policy decision point. The framework calculates a composite Trust Score from multi-dimensional risk probabilities, identity confidence metrics, and SHAP-based feature contributions to dynamically enforce context-aware access policies. Evaluated on the CERT Insider Threat Dataset r4.2 via log replay simulation, the hybrid model achieves an F1-macro score of 0.9983 and an Area Under the Receiver Operating Characteristic (AUC-ROC) of 1.0000 in identifying malicious sessions. Crucially, the system generates detailed SHAP-based reason codes to provide Explainable AI (XAI) security auditing at the access decision point. This framework demonstrates that integrating explainable anomaly detection with NIST SP 800-207 Zero Trust Architecture (ZTA) principles significantly mitigates insider risks without disrupting developer workflows.
