Insider threats targeting proprietary software repositories represent a severe risk to organizational intellectual property, which traditional boundary-focused security models fail to adequately address. To mitigate this vulnerability, this paper introduced a Zero Trust-aligned framework integrating a two-stage hybrid machine learning pipeline and SHAP-based explainability into a dynamic access control architecture.

The empirical evaluation of this framework yielded three significant results. First, the hybrid classifier achieved an F1-macro score of 0.9983 and an AUC-ROC of 1.0000 on the CERT Insider Threat Dataset r4.2, outperforming baseline models. Second, the sensitivity analysis demonstrated that a behavioral-heavy weight configuration ($w_1 = 0.4, w_2 = 0.4, w_3 = 0.2$) minimized false positives while maintaining a high True Positive Rate of 100.00%. Third, the explanation engine successfully identified and rendered the top three driving features for anomalous sessions within 25 milliseconds, providing real-time auditability.

Architecturally, this work successfully demonstrated an end-to-end integration mapping real-time behavioral telemetry to dynamic policy enforcement decisions. By utilizing Keycloak as the PEP and FastAPI as the PDP, the prototype proved that machine learning outputs can be directly translated into dynamic, database-driven policy actions such as step-up authentication or access restrictions. Section VI details key areas for future work, focusing on live agent-based endpoint telemetry and multi-tenant scaling. Ultimately, this research provides a scalable, explainable path forward for software organizations seeking to implement NIST SP 800-207 Zero Trust principles to protect high-value codebases from insider threats without compromising developer productivity.
